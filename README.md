## Project Overview

This repository is my Digital Literacy Portfolio, submitted as part of the CSE0001 course at VIT Bhopal University. In this project, I have taken on the role of a Student Digital Ambassador and completed five tasks covering different areas of digital literacy — from creating awareness content to staying safe online.

All tasks, screenshots, and write-ups are organised into separate folders as listed below.



## Task Summaries

### Task 1 – Digital Literacy Infographic
Created a one-page infographic using **Canva** covering what digital literacy is, safe internet practices, and professional online presence. Exported as an image and added to the repository.

### Task 2 – Student Digital Portfolio
Set up profiles on **GitHub**, **LinkedIn**, and **Kaggle**. GitHub profile README includes name, branch, year, and learning goals. LinkedIn Education section is filled in. All screenshots are saved in the task-2-portfolio/ folder.

### Task 3 – Coding & Collaboration Platforms
Completed a beginner challenge on **HackerRank** (Solve Me First). Created a 5-question **Digital Literacy Awareness Quiz** using Google Forms, linked to a Google Sheet for responses.

> 📋 **Google Form Quiz Link:(https://docs.google.com/forms/d/e/1FAIpQLSeMY_eQuceF37ihpOvNY34WdttcXsG9jGANJMwbTLmjjv6q-Q/viewform?usp=publish-editor\)

### Task 4 – Professional Email & Etiquette
Wrote two professional emails (assignment extension request + internship application). Created a Social Media Do's and Don'ts checklist for college students.

### Task 5 – Cybercrime Awareness
Wrote a 200+ word case study on **UPI/Online Payment Fraud**. Created an 8-point 'Stay Safe Online' prevention checklist including cybercrime.gov.in and Helpline 1930.
